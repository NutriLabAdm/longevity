# Разметка перевода для Diploma Viewer

## Онлайн загрузка

Откройте: https://longevity.startupassist.ru/pdf_viewer_translator.html

Там можно:
1. Загрузить PDF файл (кнопка "📄 PDF")
2. Загрузить файл перевода с метками (кнопка "📝 Перевод")
3. Страницы синхронизируются автоматически
4. Нажмите **S** для вкл/выкл синхронизации

## Формат файла

Текстовый файл с переводом должен быть размечен специальными тегами.

## Теги

### === PAGE N ===

Обозначает начало страницы N (1-45). Все содержимое после этого тега относится к этой странице до следующего тега PAGE.

```
=== PAGE 1 ===
Содержимое страницы 1...

=== PAGE 2 ===
Содержимое страницы 2...
```

## Пример файла

```
=== PAGE 1 ===
Магистерская диссертация

Образовательная программа: Науки о жизни
Студент: Максим Ковалев
Научный руководитель: Екатерина Храмеева
Москва, 2024

=== PAGE 2 ===
АННОТАЦИЯ

Микроглия является ключевой популяцией клеток головного мозга...

=== PAGE 3 ===
ОГЛАВЛЕНИЕ

Введение......................................................................................................................................5
Обзор литературы............................................................................................................................8

=== PAGE 4 ===
ВВЕДЕНИЕ

Нейродегенеративные заболевания, включая болезнь Альцгеймера (БА)...
```

## Как сгенерировать HTML

```bash
cd E:\_dev\55.Skoltech\55.BMEI-2025\data
python generate_viewer.py
```

Скрипт `generate_viewer.py`:
1. Читает файл `translation_marked.txt`
2. Парсит теги `=== PAGE N ===`
3. Генерирует HTML с двумя колонками

## Важно

- Теги должны быть в формате `=== PAGE N ===` где N - число
- Между тегами может быть любое количество абзацев
- Абзацы разделяются пустой строкой (двойной ENTER)
- HTML автоматически оборачивает каждый абзац в тег `<p>`
- Не удалять на хостинг - только генерировать локально

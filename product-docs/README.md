## Product Overview

Healora AI is a digital twin for obesity treatment and longevity. It uses AI to analyze multi-omics data, create personalized models, and generate longevity protocols.

## Key Features

1. **Data Upload Module** - Multi-source data ingestion
2. **Digital Twin Engine** - Hybrid mechanistic/ML modeling
3. **Longevity Path Generator** - Personalized recommendations
4. **Tracking & Optimization** - Continuous monitoring

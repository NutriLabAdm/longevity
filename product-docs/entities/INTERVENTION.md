## 1. Core Behavioral Science Framework

### 1.1 Habit Loop Theory

Every behavior follows the Habit Loop:

| Component | Description | Example |
|-----------|-------------|---------|
| **Cue** | Trigger that initiates behavior | 9 PM, stress, social situation |
| **Behavior** | The action or routine | Eating, scrolling phone, exercise |
| **Reward** | Benefit the brain perceives | Dopamine hit, stress relief, pleasure |

**Intervention Tactics:**
- **Cue redesign** — Remove or modify triggers
- **Behavior substitution** — Replace unwanted with wanted
- **Reward recalibration** — Shift perceived benefits

### 1.2 Stages of Behavior Change (Transtheoretical Model)

| Stage | User State | HEALORA Response |
|-------|-------------|-------------------|
| **Precontemplation** | Not aware of problem | Educational content, awareness prompts |
| **Contemplation** | Considering change | Cost-benefit analysis, motivation |
| **Preparation** | Planning to act | Specific action plans, resource prep |
| **Action** | Actively changing | Daily micro-quests, reinforcement |
| **Maintenance** | Sustaining change | Long-term tracking, relapse prevention |

## 5. Behavior Change Strategies

### 5.1 Environmental Design

| Strategy | Implementation |
|----------|----------------|
| **Cue Removal** | Remove triggers for unwanted behaviors |
| **Cue Addition** | Add visual reminders for wanted behaviors |
| **Friction Increase** | Make unwanted behaviors harder |
| **Friction Reduction** | Make wanted behaviors easier |
| **Social Design** | Build accountability into environment |

**Examples:**

| Unwanted Behavior | Environmental Fix |
|-----------------|-------------------|
| Late-night snacking | Hide snacks, lock kitchen after 8 PM |
| Morning screen time | Leave phone in other room |
| Skipping workout | Lay out clothes night before |
| Overeating | Use smaller plates, pre-portioned meals |

### 5.2 Implementation Intentions

**Format:** "If [situation], then [behavior]"

**Examples:**
- "If I feel stressed after work, then I'll take 5 deep breaths"
- "If I'm hungry between meals, then I'll drink water first"
- "If it's raining, then I'll do indoor stretching"

### 5.3 Social Accountability

| Feature | Description |
|---------|-------------|
| **Accountability Partner** | One person shares goals and check-ins |
| **Coach Connection** | Professional guidance on progress |
| **Community** | Group challenges and shared journeys |
| **Team Challenges** | Workplace or social group competitions |

### 5.4 Stress Mindset Intervention

**Tracked Metrics:**
- Stress perception (threat vs challenge)
- Stress response (avoidance vs approach)
- Recovery from stress (time to baseline)

**Intervention Prompt:**
> "How do you view this stressful situation?"
- [ ] Threat - I can't handle this
- [ ] Challenge — This is an opportunity
- [ ] Neutral — Just a normal part of life

|---------|-----------------|-------------------|
| **The Procrastinator** | Delay, avoidance | Micro-quests, friction reduction |

### 7.2 Intervention Selection Algorithm

```
1. Analyze behavior data
2. Identify pattern (triggers, rewards)
3. Match to segment
4. Select lowest-friction intervention
5. Test for 14 days
6. If no improvement, escalate to next intervention
7. Track and optimize
```

## 8. Recovery & Relapse Prevention

### 8.1 Lapse Response Protocol

| Stage | User Action | HEALORA Response |
|-------|-------------|-------------------|
| **Minor slip** | 1 missed day | "One miss doesn't break your streak" |
| **Moderate lapse** | 2-3 missed days | "Let's restart small — just log today" |
| **Major relapse** | 7+ missed days | Full reassessment, simpler goals |

### 8.2 Re-Engagement Tactics

- **Non-judgmental language** — "Here to help, not to judge"
- **Reset progress bar** — Visual fresh start
- **Memory lane** — Show past successes
- **Simpler entry** — Start with one behavior
